All programming languages follow the same set of operators mentioned in [here](Operators). 
The precedence is also same, some might introduce something new, but these are the basic elements for all programming languages and hence is the same.


---
# What is new?
- No pointer or pointer arithmetic – Java disallows these for security and reliability.
- All arrays and strings are objects, not primitive types or pointers.
- Boolean expressions _must_ be written as actual boolean logic, not integer “truthy” values.
