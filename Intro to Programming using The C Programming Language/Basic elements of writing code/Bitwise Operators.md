A bitwise operator is an operator that acts on the individual bit on the binary number. Recall that a bit is a digit of a binary number.

Since the computer stores everything in form of binary, it is generally fast to perform bitwise operations to achieve a task.
A bit is called *set* if it is $1$, and *free* if it is $0$.
