# What is the use?
The `<span>` tag is a generic inline container used in HTML to group and style specific portions of text or inline elements within a document. 
Unlike block-level elements such as `<div>`, the `<span>` tag does not introduce line breaks or disrupt the flow of content—it simply wraps the targeted content so you can apply styles or behaviors to it without affecting the entire block
