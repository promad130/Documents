# What is the use?
The `<img>` tag is used to embed images into an HTML document. It is an **empty (self-closing) inline element**, meaning it does not require a closing tag and does not start on a new line

## When to use?
Whenever you want to add an image.

## Basic Syntax
```xml
<img src="image.jpg" alt="Description of image">
```
### **Required Attributes**
- **src**: Specifies the path (URL) to the image file. This can be a relative path, absolute path, or an external URL.
- **alt**: Provides alternative text for the image, shown if the image fails to load and used by screen readers for accessibility
