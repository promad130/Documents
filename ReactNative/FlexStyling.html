Interactive Flexbox Guide

Flexbox is a "one-dimensional" layout model. It deals with layout in one direction at a time — either as a row or a column.

The Two Fundamental Axes

The most important concept is the relationship between the Main Axis and the Cross Axis.

  • Main Axis: The primary axis (defined by flex-direction).
  • Cross Axis: Runs perpendicular to the main axis.

The Power of Shorthand: flex: 1

In real projects, you rarely set grow, shrink, and basis separately. Instead, we use the flex property.
flex: 1; is shorthand for flex: 1 1 0%;

What does it actually mean?

It tells the item: "Grow to fill available space, shrink if you have to, and start with a base size of 0 (ignore my content size)."

  • flex: initial (0 1 auto) - Doesn't grow, can shrink.
  • flex: auto (1 1 auto) - Grows and shrinks based on content.
  • flex: none (0 0 auto) - Fully rigid.

Individual Component Properties

flex-grow

Defines the ability for a flex item to grow. It's a proportion of the remaining space.

flex-shrink

Defines the ability for an item to shrink when there isn't enough space.

flex-basis

The default size of an element before the remaining space is distributed.

Summary Checklist

  1. Parent: Use display: flex to start.
  2. Direction: Decide if you want a row or a column.
  3. Main Axis: Use justify-content for spacing.
  4. Cross Axis: Use align-items for alignment.
  5. Shorthand: Use flex: 1 to make an item fill available space.
