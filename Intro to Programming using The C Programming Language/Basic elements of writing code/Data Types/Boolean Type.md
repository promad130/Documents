Boolean data type is used to give a clear to the point result, i.e, either true or false.

Hence, it stores only two values, i.e., 0 or 1, where 0 refers to false, and everything except 0 directly points to true. This concept is used in programming languages as well, in order to represent true or false without creating a boolean variable.
